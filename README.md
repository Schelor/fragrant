## 用户指南
Guava项目包含很多谷歌的核心库，我们在项目中用到了很多这些Java类库，如集合、缓存、基础类型支持、并发库、通用注解、字符串处理、I/O等。Google的码农每天都在使用这些工具，并切已用之于生产服务中。

查阅Javadoc不总是最有效的方式来学习以及使用一个类库。对于Guava，我们尝试提供可读性良好的，令人愉悦的说明，来解释很受欢迎，实用性很强功能特性。
